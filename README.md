# dotfiles

```zsh
chmod +x install.sh
./install.sh
```

## Applications

- Ghostty - <https://ghostty.org/> terminal.
- Rectangle - <https://rectangleapp.com/> workspace and window arrangement.
- Pastebot - <https://tapbots.com/pastebot/> multi clipboard.
- sdkman - <https://sdkman.io> SDK manager.
